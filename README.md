# Proyecto Final: Gestión de Convenios

Este proyecto se realizó para simplificar y optimizar la administración de acuerdos y contratos que realizan las Universidades. 

## Herramientas que se utilizaron:
- Node.js
- React.js
- Javascript
- MySQL

## Contenido:

- Actividades
- Convenios
- Tipo de Convenios
- Organismos
- Tipo de Organismos
- Responsable
- Resolución
- Usuarios
- Roles


### Usos:
Para administrarlos y optimizar los distintos contenidos, se podrá:
- Añadir
- Editar
- Dar de Alta
- Dar de Baja
